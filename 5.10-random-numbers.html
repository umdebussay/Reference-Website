5.10 Random Numbers

Notes

Intro

Random numbers are useful in software development for many reasons - game mechanics, generating usernames/passwords, anytime you need something seemingly unpredictiable.

Pseudo-Random Numbers - computers are just big calculators solving formulas and cant create truly random numbers, but instead use an algorithm that generates numbers that seem random.

Random numbers in JS

JavaScript provides a number of functions that are useful for generating random numbers.

  • Math.random() is a function that returns a pseudo-random number between 0 and 1, not including 1.
  • Math.floor() if a function that takes a decimal number and rounds it down - chops off the decimal part.

    • Random number calculations

    To produce a random integer from a min value to a max value(uncluding the min and max) use calculations.

  • Math.floor(Math.random() * (max +1)) + min
    • Example

    Generate a random number from 1 to 10 and store it in a variable

    var randNum = Math.floor(Math.Random) * 11) + 1;
