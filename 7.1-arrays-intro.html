7.1 Intro to Arrays and Lists

Objectives

To be able to declare an array with values
To be able to access and assign elements to an array using subscript [ ] notation

Notes

Variables

Recall: Variables are named locations in memory that we can store data in - essentially a box with a name we can put something in.
We've been limited though, because a variable can only store one piece of data at a time. If we needed to store 12 names, we would need to create 12 variables. One to hold each name.


Arrays and Lists

An array or list solves that problem by allowing us to create one variable that holds multiple items.

Definition
An array is an ordered collection of data. The data can be any data type (int, float, string, boolean, or object). Each item in the array is numbered based on its position within the array, starting with zero. This number is called its index

Syntax examples to declare array variables (remember, declare means to create)

var arr = ["first item", "second item", "third item"]; // An array with three strings
var names = ["John", "Jacob", "Jinglheimerschmidt"]; // An array with three strings 
var nums = [10, 12, 0, -15]; // An array with 4 ints
The square brackets [] tell the interpreter to create an array variable. Each item in the array is separated by a comma.
We often use plurals to name array variables because they often hold multiple, similar types of data ex. names for a list of names.

Array Terminology

  • Array or list: the data type that can hold an ordered list of values. In JavaScript, arrays and lists are the same thing. In other programming languages array is the basic version of
  • Element or item: the value of one item in the array
  • index: the numerical position of an element in the array with 0 being the index of the first element
  • length: the number of elements in the array
Example
var cities = ["Tucson", "Phoenix", "Mesa", "Flagstaff"];
cities is the name of the variable that holds the array
cities has a length of 4.
The first element in the array is "Tucson". The index of "Tucson" is 0

Accessing and Assigning Values in an Array

To access (get the value) and assign (change the value) in an array, we use subscript or square bracket [] notation to specify the index we want to access or assign to.

Example 

// Declare an array of student names
var topStudents = ["Christian", "Kristian", "Shakka", "Jaime", "AJ"];
// Access the first element to store in a variable (index 0)
var student = topStudents[0];
// Assign the third element (index 2) to be "Alexa"
topStudents[2] = "Alexa";
