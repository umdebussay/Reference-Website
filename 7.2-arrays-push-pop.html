7.2 Adding to and Removing from Arrays

Objectives

To be able to use the .push() and .pop() methods of an array to add and remove elements from the array

Warm-up

  1. Create a global variable called favFoods and assign it to be an array of strings that are your top three favorite foods.
  2. Use console.log() to print out each element to the console individually

Notes

Arrays are Objects

  • An array is a JavaScript object
  • Array objects have built-in properties and methods to help us work with the array
  • Array properties will let us programmatically determine the length, add elements, remove elements, find elements, sort elements, and more

Array Length

When you declare an array, it has a fixed length 

var favFoods = [“Pizza”, “Burgers”, “Ice cream”];
// favFoods has a length of 3
var arr = [];
// arr has a length of 0
  • You can get the length of an array programmatically using the .length property
  • Array indices go from 0 to arr.length - 1
  • If you try to access an element at an index outside this range, the element will be undefined (this is a semantic or logic error)
  • See example in .js file

Adding elements to an array

There are several ways to add elements to an array. The most basic way is to use the .push(newElement) method. The .push() method adds the element to the end of the array, increasing its length by 1. The .push() method returns the new length of the array.

Example (see .js file) 

var arr = [“red”, “green”, “blue”];
arr.push(“orange”);

Removing elements from an array

There are several ways of removing elements as well. The most basic way is to use the .pop() method. The .pop() removes the last element from the array and returns it.

Example (see .js file) 

var arr = [“red”, “green”, “blue”];
var lastElem = arr.pop();
After this code runs, arr will contain “red” and “green” and lastElem will contain “blue”
Note: You do not have do anything with the element returned by the .pop() method. You might just want to let it disappear effectively deleting it from the array
