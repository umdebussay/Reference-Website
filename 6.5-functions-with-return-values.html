6.5 Function Return Values

Objectives

  • To be able to define a function that calculates and returns a value

Notes

Function Return Values

Parameters are a way of passing data into a function. Return values are a way for functions to pass data back to the point the function was called at.

Syntax
return value; where value is a literal string, integer, float, or boolean, expression, or a variable containing a value.

Example 

1 function start() {
2   var result = double(10);
3 }
4 
5 function double(num){
6   return 2 * num;
7 }
On line 2, double(10) jumps down to line 5 where 10 is stored in the num parameter. On line 6, num is multiplied by 2, which results in the value 20. This value is then returned to the point of the function call on line 2 and stored into the variable result.

Return statements

return value; is called a return statement. A return statement ends the function execution, even if there are lines after the return statement, and returns execution to the point of the function call. It's similar to break in a loop

Example 

1 function doSomething() {
2   var num = 20;
3   num += 50;
4   return num;
5   // The statements below will never be reached because of the return statement above
6   num = 100;
7   return num;
8   // Any thing after the line above will be skipped         
9 }

Working Example

Write a function named areaOfRect() that returns the area of a rectangle given the rectangle's length and width. Then, use the function to calculate and display using alert() the combined area of two rectangles
areaOfRect(10, 20) should return 200

Problem 1

Write and test a function named isEven() that will take an integer as a parameter and return true if the integer is even and false if it's odd. Display the result in an alert
isEven(10) should return true
isEven(231) should return false
Hint #1: You will need to use an if-else statement
Hint #2: A number is even if there is a remainder of zero when divided by two. (Look up modulus if that isn't enough of a hint)

Problem 2

Write and test a function name min() that takes two number parameters and returns whichever is smaller. Display the result in an alert.
min(10, 20) should return 10
min(57, -1) should return -1
