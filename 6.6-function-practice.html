6.6 Function Practice

Objectives

  • To be able to define functions with multiple parameters that return a value
  • To be able to name functions, parameters, and variables appropriately for a given situation
  • To practice thinking algorithmically

Directions

  1. Solve each challenge in the linked .js file
  2. Use comments to label each challenge ex: // Problem #1
  3. Test your functions by calling them and using console.log() to print the output

Problem #1

Write a function that will convert a number of minutes to seconds

minToSec(20) -> 1200

Problem #2

Write a function that will take two numbers as parameters and return true if the sum of the numbers is less than 100.

lessThan100(10, 20) -> true
lessThan100(90, 20) -> false

Problem #3

Write a function that will calculate the area of a triangle given its base and height. Area = 0.5 X base X height

areaOfTriangle(6, 10) -> 30

Problem #4

Write a function that will take a number and return the next even number after it

nextEven(7) -> 8
nextEven(10) -> 12

Problem #5

Write a function that will take someone's age in years and return the equivalent number of seconds

ageInSeconds(15) -> 473040000

Problem #6

Write a function that will take a base and a power and return the base raised to that power. Use a for or while loop to solve this one.

power(10, 2) -> 100

Problem #7

Write a function that will take two numbers and return the remainder after dividing the first number by the second

remainder(7, 3) -> 1
remainder(19, 4) -> 3

Problem #8

Write a function that will take the number of 2-pointers and 3-pointers scored in a basketball game and return to total score

bBallScore(10, 7) -> 41

Problem #9

Write a function that will take the number of touchdowns (including extra point) and fieldgoals scored in a football game and return the total score

footballScore(2, 1) -> 17

Problem #10

Write a function that will take the number of chickens, cows, and pigs and return the total number of legs. Chickens have 2 legs, cows 4 legs, and pigs 4 legs

countLegs(3, 1, 2) -> 18
